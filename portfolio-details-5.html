Project details

This project is built using ReactJS, Ruby on Rails, TailwindCss, PostgreSQL, and React Query. The project is a bug tracker that allows users to create, edit, and delete bugs. React Query manages fetched state while also making the application more performant by using cached data. Global props were achieved using the useContext hook and allows the state to be shared across the application. The back-end also uses BCrypt for password encryption.
